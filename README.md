# numblr

Alternative Tumblr (and Twitter, Instagram, RSS, ...) frontend.

Inspired by [nitter](https://github.com/zedeus/nitter).

## Features

- ✓ rss and atom
- ✓ tumblr (via rss)
- ✓ twitter (via nitter)
- ✓ instagram (via bibliogram)
- ✓ mastodon (via rss)
- ✓ in-memory cache
- ✓ native dark mode
- ✓ settings stored in cookie
- ✓ lists
